# Write target-aligned uncertainty tables

Write target-aligned uncertainty tables

## Usage

``` r
write_gazepoint_target_uncertainty(
  x,
  directory,
  prefix = "gazepoint_target_uncertainty",
  overwrite = FALSE
)
```

## Arguments

- x:

  A gp3ml uncertainty object.

- directory:

  Output directory.

- prefix:

  Filename prefix.

- overwrite:

  Whether existing files may be replaced.

## Value

Named paths, invisibly.
