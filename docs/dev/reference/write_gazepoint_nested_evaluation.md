# Write nested-resampling evaluation tables

Write nested-resampling evaluation tables

## Usage

``` r
write_gazepoint_nested_evaluation(
  x,
  directory,
  prefix = "gazepoint_nested_evaluation",
  overwrite = FALSE
)
```

## Arguments

- x:

  A `gp3ml_nested_evaluation`.

- directory:

  Output directory.

- prefix:

  Filename prefix.

- overwrite:

  Whether existing files may be replaced.

## Value

Named paths, invisibly.
