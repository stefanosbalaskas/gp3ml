# gp3ml: Leakage-Safe Predictive Validation for Gazepoint Research

`gp3ml` provides governance and validation infrastructure for predictive
workflows using Gazepoint-derived research data.

## Details

The initial development scope is limited to leakage protection, role
validation, feature provenance, group-aware validation contracts, and
reproducibility reporting.

The package does not treat gaze, pupil, or biometric measurements as
direct indicators of emotion, stress, mental state, cognition,
comprehension, personality, diagnosis, deception, or intent.
