# Write a model card

Write a model card

## Usage

``` r
write_gazepoint_model_card(
  card,
  path,
  format = c("markdown", "json"),
  overwrite = FALSE
)
```

## Arguments

- card:

  A `gp3ml_model_card` object.

- path:

  Destination file path.

- format:

  Output format: Markdown or JSON.

- overwrite:

  Whether an existing file may be replaced.
